# Поехали в Крым — маршруты по ЮБК

Первая версия (preview) мобильного PWA-гида по маршрутам Южного берега Крыма: прогулки, автосценарии, парковки, предупреждения и точки для навигатора.

Стек: React 19 + Vite 6 + TypeScript + Tailwind CSS 4.

## Что сейчас в MVP
- 10 готовых маршрутов, 12 — «скоро»
- Страницы маршрутов: план, парковка, предупреждения, точки с координатами
- Открытие точек в Яндекс Картах / Google Maps / 2ГИС, копирование координат
- Фильтры и подбор маршрута, закладки (localStorage)
- Блок «Актуальная обстановка» + форма «Сообщить обстановку» (демо: сохраняется локально)
- PWA: установка на главный экран, базовый офлайн-фолбэк

## Что пока не реализовано
- Реальная погода (показываем честный бейдж «проверьте перед выездом»)
- Общий backend для пользовательских отчётов (есть опциональный Supabase-хук)
- Офлайн-карты

## Запуск
```bash
npm install
npm run dev        # http://localhost:3000
```

## Сборка и проверка
```bash
npm run build      # dist/
npm run preview    # локальный просмотр сборки
npm run lint       # tsc --noEmit
```

## Деплой
**Vercel:** импортируйте репозиторий — `vercel.json` уже настроен (build `npm run build`, output `dist`, SPA-rewrites). Если корень проекта — папка `app`, укажите её как Root Directory.

**Netlify:** build command `npm run build`, publish directory `dist`. SPA-redirect лежит в `public/_redirects`.

**Статичный хостинг / GitHub Pages:** сборка использует относительные пути (`base: './'`) — содержимое `dist/` работает из любой подпапки.

## Обратная связь (настройка)
Кнопка «Сообщить о правке» использует `src/config/contacts.ts`: укажите `FEEDBACK_EMAIL` и/или `TELEGRAM_URL`. Пока оба пусты, кнопка честно копирует готовый шаблон правки в буфер — без фейковой «отправки». Форма на экране «Доступ» показывается только при настроенном Supabase; без него — кнопка «написать нам».

## Важно
Это preview для тестирования. Данные маршрутов ориентировочные: перед поездкой проверяйте погоду, состояние дороги, доступ и режим объектов — особенно в горах, заповедниках и на серпантинах.
